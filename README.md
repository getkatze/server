# Katze Swag Server

## This is the swaggiest hackathon server in the entire west

### Our Stack

- Typescript
- ApolloServer
- Express
- Postgresql (Prisma)
- Graphql (Type-Graphql + Prisma type generators)

## Using

#### Migrations

```shell 
yarn prisma migrate dev
```

#### Push db

```shell
yarn prisma db push --preview-feature
```

#### Formatting

```shell
yarn format
```

#### Running

```shell
yarn start
```
